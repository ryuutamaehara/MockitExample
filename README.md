# MockitExample

## Version
java 1.8.0_19  
junit 4.13  
mockito 1.10.19

## Overview
Mockitoによるmockの動作を確認するためのリポジトリです。  
MockTest.javaの中には次のサンプルメソッドが含まれます。

- モックオブジェクトの作成  
- スタブメソッドの定義  
- スタブメソッドの検証  
- 例外をスローするスタブメソッドの定義。  
- voidを返すスタブメソッドの定義  
- 任意の引数のスタブメソッドの定義  
- 部分的モックオブジェクトの作成。  
- スパイオブジェクトの作成。
